# ✨ Jules Accessory

### Elegância em cada detalhe 💎

Website desenvolvido para a marca **Jules Accessory**, focado em apresentar acessórios e bijuterias de forma moderna, sofisticada e responsiva.

</div>

---

# 📖 Sobre o Projeto

O projeto foi criado com o objetivo de transmitir a identidade visual da marca através de um design elegante, delicado e minimalista.

A aplicação conta com:

- Página inicial moderna
- Catálogo de produtos
- Sessão sobre a marca
- Integração com Instagram
- Layout totalmente responsivo
- Navegação suave entre seções
